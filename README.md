# Portfolio Website 🚀

This is my professional portfolio website built to showcase my projects and technical skills. It features a fully functional admin dashboard for content management.

## 🛠 Tech Stack
- **Framework:** Laravel 12
- **Frontend:** Vue.js 3
- **Styling:** Tailwind CSS
- **Database:** MySQL
- **Image Processing:** Intervention Image

## ⚡ Features
- **Admin Dashboard:** Secure login system.
- **Project Management:** Create, Read, Update, and Delete projects.
- **Skill Proficiency:** Track technical skills with visual progress bars.
- **Responsive Design:** Fully responsive layout for all devices.
